# Wordle

Copy of the Wordle game, where the goal is to figure out the random word through trial and error.

# Site to test

https://wordle-kaua.netlify.app/


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
